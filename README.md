A tiny pink diary-style web app made for Princess Day (Nov 18).

- Secret word gate (she must type **EDIL** to unlock)
- Upload a portrait
- Tap to place a falling crown
- Choose a theme: Belle / Barbie / Kitty
- Download the crowned portrait as an image

Built with Next.js 14, React, TypeScript and Tailwind CSS.

## Getting started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the dev server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

### 3. Build for production

```bash
npm run build
npm start
```

## Deploying

You can deploy directly to Vercel (recommended). See the main instructions in the ChatGPT message.
